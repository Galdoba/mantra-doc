## Категория: ЗАДАЧА
Написать и настроить поддержку программы для валидации и исправлений субтитров.

### **Описание:** 
mantra-srt-validator - утилита для работы с субтитрами, является частью экосистемы mantra.

#### Принцип работы программы:
в качестве аргументов принимаем именна/пути srt файлов. Программа создает копию исходного файла, читает содержание, валидирует по формату subrip, вносит правки по содержанию, сохраняет копию обработанного файла и перезаписывает исходный файл обработаным.

#### Основные команды программы:
```bash
#пустая, рутовая команда
mantra-srt-validator [args]
запускает основной алгоритм обработки субтитров
args - имена/пути afqkjd для обработки

--/--/-- revert [args]
восстановление исходного файла из хранилища

--/--/-- reload [args]
восстановление обработанного файла из хранилища
```
