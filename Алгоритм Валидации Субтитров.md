### **Описание:** 
1. Парсинг байтов
	1. Проверка кодировки
	2. Проверка формата данных
2.  Оценка последовательности Таймкодов
	1. Наличие гапов
	2. Инкрементальность
3. Оценка Содержания контента
	1. Подтверждение того что все символы допустимы
	2. Подтверждение количества строк
4. Оценка длительности
	1. Группируем титры по 1+ штук. (если Гап между титрами менее 2 секунд, считаем их одной группой) 
	2. Распределяем веса изменений:
		1. Если титр меньше 7000 мс = -1
		2. Если титр больше 1000 мс = +1
		3. Если cps > 23 = +1
		4. Если cps < 12 = -1
		5. Если wpm < 120 = -1
		6. Если wpm > 150 = +1
		7. Если смещение > 1 = двигаем влево (-1)
		8. Если смещение < -1 = двигаем вправо (+1)
		9. Если Нахлест - раздвигаем оба титра по разным сторонам
	3. применяем сумму векторов ко всем началам и концам
	4. Проверяем на ошибки:
		1. Нахлест
		2. Вектор не 0
