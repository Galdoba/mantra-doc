# Storage
Корабли и техника имеют внутренне пространство для хранения расходников (подробные механики смотри в [[Supply Unit (SU)]], [[Life Support Supplies (LSS)]], [[Materials]]):
- Корабль (Ship): 100 x Тоннаж Корабля.
- Техника (Vechicle): 50 x Space (размер техники).

Если нужно хранить больше расходников можно использовать трюм.

|          Type          | Cargo<br> | Space | Base Cost | Usage        |
| :--------------------: | :-------: | :---: | :-------: | ------------ |
|           SU           |   0.01    | 0.05  |   Cr100   | maintainance |
|          LSS           |   0.04    |  0.2  |   Cr150   | maintainance |
| Materials (Structural) |    0.1    |  0.5  |  Cr10000  | repair       |
| Materials (Technical)  |   0.01    | 0.05  |  Cr25000  | repair       |
| Materials (Outfitting) |    0.1    |  0.5  |  Cr1000   | repair       |
