## Категория: РЕШЕНИЕ
Перебираем варианты BOM исходя из кодировки и отрезаем первые байты при совпадении.

### **Описание:** 

##### Варианты BOM для кодировок Unicode:

| Кодировка      | Порядок байтов | Шестнадцатеричная последовательность BOM                                          |
| -------------- | -------------- | --------------------------------------------------------------------------------- |
| **UTF-8**      | —              | `EF BB BF`                                                                        |
| **UTF-16**     | Little-endian  | `FF FE`                                                                           |
| **UTF-16**     | Big-endian     | `FE FF`                                                                           |
| **UTF-32**     | Little-endian  | `FF FE 00 00`                                                                     |
| **UTF-32**     | Big-endian     | `00 00 FE FF`                                                                     |
| **UTF-7**      | —              | `2B 2F 76` (варианты: `2B 2F 76 38`, `2B 2F 76 39`, `2B 2F 76 2B`, `2B 2F 76 2F`) |
| **UTF-1**      | —              | `F7 64 4C`                                                                        |
| **UTF-EBCDIC** | —              | `DD 73 66 73`                                                                     |
| **SCSU**       | —              | `0E FE FF`                                                                        |
| **BOCU-1**     | —              | `FB EE 28`                                                                        |

#####  Пример golang utf8:
```go

utf8BOM := []byte{239, 187, 191}
if data[0:3] == utf8BOM {
	data = data[3:]
}

```
