UWP - это короткая строка, содержащая коды характеристик [[World|мира]]. Каждый символ строки представляет собой код [[ehex]]. Коды дают общее представление о характеристиках, но их можно развернуть для получения дополнительных деталей.

# Схема
Пример UWP: `A123456-7`, где:
- `A`= код космопорта
	- Возможные значения: `ABCDEFGHYX`
- `1`= код размера
	- Возможные значения: `0123456789ABCDEFRS`
- `2`= код атмосферы
	- Возможные значения: `0123456789ABCDEFGH`
- `3`= код гидросферы
	- Возможные значения: `0123456789A`
- `4`= код населения
	- Возможные значения: `0123456789ABC`
- `5`= код правительства
	- Возможные значения: `0123456789ABCDEF`
- `6`= код законов
	- Возможные значения: `0123456789ABCDEFGHJKLMN`
- `-`= сепаратор
- `7`= код технологического уровня
	- Возможные значения: `0123456789ABCDEFGHJKLMNPQRSTUVWXYZ`

# Как объект-значение
UWP представляет собой самогенерирующийся объект. Для его построения не нужны внешние данные. Броски кубиков реализованы в [[Эмулятор игральных костей]]
