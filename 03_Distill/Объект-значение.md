Объект-значение (valueObject) — это объект, который можно идентифицировать по составляющим его значениям. Объект-значение может инкапсулировать логику анализа значения, его проверки и извлечения различных атрибутов. Такие объекты устраняют необходимость в соглашениях, например необходимость помнить, что эта строка является адресом электронной почты, а вот эта строка — номером телефона

Объекты-значения реализуются как неизменяемые (**immutable**) объекты. Изменение одного из полей объекта значения концептуально создает другое значение — **другой экземпляр объекта-значения**. Равенство объектов-значений основано на составляющих их значениях

Объекты-значения следует использовать при любой возможности.

Полезно использовать объекты-значения для элементов предметной области, описывающих свойства других объектов. В частности, это касается свойств [[Entity|сущностей]].

Объекты-значения инкапсулируют инварианты, *уменьшая таким* ***образом сложность*** и  [[Степень свободы]]
