## Категория: ЗАДАЧА
Валидация доменного объекта subrip

### **Описание:** 
Требуется валидация отношений каждой записи по следующим пунктам:
- [+] Индексация должна начинаться с единицы и увеличиваться строго на 1
- [+] Конечное время должно быть больше начального
	- [+] Минимум 1.5 секунд
	- [+] Максимум 6.5 сукунд
- [+] Любой таймкод не-первой записи должен быть больше конечного таймкода предыдущей записи
- [ ] Текст субтитра не должен содержать Проблемных глифов
- [ ] Текст субтитра не должен содержать Омоглифов
- [ ] Текст субтитра должен быть разделен только один раз
	- [ ] Разделяем по последнему пробелу в позиции 34-46
	- [ ] Предпочтение пробелу после глифа пунктуации
