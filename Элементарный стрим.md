Элементарный стрим - поток данных в файле содержащий медиа-информацию.

# Типы стримов
## Видео
Содержит последовательность кадров имеющих единое кодирование. Чаще всего присутствует только один для видео файла. Если больше, то это [Сложный случай].
## Аудио
Содержит последовательность аудио-сэмплов имеющих единое кодирование. Может отсустствовать или присутствовать во множественном колличестве.
## Данные
Содержит техническую информацию. Чаще всего не используется.
## Субтитры
Содержит субтитры для аудио. Чаще всего игнорируется, но может понадобится конвертировать есго в [Файл субтитров].
