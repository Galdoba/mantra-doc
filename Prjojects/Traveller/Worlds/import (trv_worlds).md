import - команда формирует серию запросов на https://travellermap.com для сбора радиальных регионов начиная с точки {0,0}(Reference) размером в 12 парсек. установленный радиус указывает какое количество кругов нужно пройти (сколько данных собрать)

Собранные данные пакуются в формат [[t5ss]] и записываются в [[External_DB]] по ключу глобальных координат. 

# Правила работы с данными.
- Запись в базу производится только этой командой.
- Данные статичные и не дополняются другими методами.
- Данные хранятся в виде примитивов (строка и число)
