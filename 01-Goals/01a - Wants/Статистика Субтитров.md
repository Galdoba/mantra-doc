Имеет смысл держать в хранилище N последних отчетов (статистик) оцененных субтитров.
Храним как map[string]Stats - при повторном сканировании старое значение статистик перезатирается.

```go
type Stats struct {
	SourceName string //ключ-название файла
	WordCount  int    //счетчик слов
	Cyrilic float64 //процентное соотношение кириллических символов от общего
	Latin float64 //процентное соотношение латинских символов от общего
	Issures []string//список найденных проблем и иных тонких мест
	
}
```
общим считаем все не пробелы, не дефисы и не знаки припинания
знаки припенания стоят между буквой и пробелом
