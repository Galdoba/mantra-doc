Пакет описывающий объект-значение "Орбита"

```go
type Orbit struct {
	Distance float64 //Orbit# - для орбиты вокруг звезды; PD - для спутников планет
	Eccentricity float64 //эксентириситет, позволяет высчитать ширину (для поясов), high и low (верхнии/Нижнии границы)
	Period float64 //период обращения в часах - высчитывается отдельной функцией
}
```

Сам объект служит ключем к карте тел в звездной системе и является частью профайла ([[Star System Profile]])
