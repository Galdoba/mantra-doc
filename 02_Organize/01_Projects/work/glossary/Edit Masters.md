**Мастеры для монтажа (Edit Masters, Production Masters)** — это специально подготовленные медиафайлы, полученные путём транскодирования исходных материалов в производственный [[Кодек|кодек]] и [[Контейнер|контейнер]], которые служат непосредственной рабочей основой для творческого процесса сборки, цветокоррекции и обработки видеоконтента.

### Ключевые атрибуты:

1.  **Рабочий формат:** Кодек визуально без потерь (visually lossless) или сжатия без потерь (lossless), оптимизированный для многократной декомпрессии и обработки в реальном времени (например, **Apple ProRes**, **Avid DNxHR/DNxHD**, **CineForm**).
2.  **Функциональное назначение:** Обеспечивают плавный предпросмотр, точное позиционирование на временной шкале и высокое качество изображения для применения эффектов и цветокоррекции, выступая цифровым эквивалентом монтажного негатива.
3.  **Техническая нормализация:** Приведение разнородных исходников (с разных камер, с различными кодеками) к единому стандарту по цветовому пространству, гамме, разрешению и частоте кадров для бесшовной работы в рамках одного проекта.
4.  **Иерархическое звено:** Занимают промежуточную позицию между «сырыми» исходниками (Camera Raw) и облегчёнными прокси-файлами, представляя собой баланс между качеством и производительностью.
