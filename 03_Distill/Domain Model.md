Модель — это упрощенное представление вещи или явления, в котором намеренно подчеркиваются одни аспекты и игнорируются другие. Это абстракция с точки зрения конкретного использования. 

У всех моделей есть цель, а эффективная модель содержит только те детали, которые необходимы для достижения этой цели. Пример: кусок картона на полу (коробка из под холодильника) - это модель холодильника с точки зрения, того какую площадь он будет занимать в проходе.
**Полезная модель — это не копия реального мира.** Она предназначена для решения конкретной задачи и должна предоставлять для этого достаточный объем информации.

Модель строится исходя из [[Ubiquitous Language|единого языка]].

Ментальные модели экспертов предметной области *могут сами по себе быть противоречивыми.* Разными экспертами предметной области **могут использоваться разные модели одной и той же сферы деятельности.**

Теоретически, хотя это и непрактично, одна модель может охватывать всю предметную область [но лучше делить?].
