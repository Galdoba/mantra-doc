Структурированный логгер - это [[Logger|логгер]], который записывает сообщения в виде текстового представления структуры поддающейся сериализации (см. [[Serialization|сериализация]]). 

## Фиксированые поля (универсальные)
Поля которые присутствуют (или могут присутствовать во всех сообщения отдельного логгера):
- message - непосредственное тело записи - то что должно быть прочитано тем кто будет читать лог
- timestamp - временная метка момента создания сообщения
- level - уровень логирования ассациированного с сообщением
- file - файл, выполнение кода которого вызвало сообщение (для TRACE/DEBUG)
- line - строка в файле, выполнение кода которого вызвало сообщение (для TRACE/DEBUG)
- funcName - имя функции, выполнение кода которой вызвало сообщение (для TRACE/DEBUG)

## Произвольные поля (контекст)
Поля которые присутствуют (или могут присутствовать только в конкретных записях):
- значение параметра
- ID структуры или строковое представление самой структуры
Эти поля должны добавляться отдельно от message. некоторые поля могут содержать чувствительную информацию и должны быть подвержены обфускации (см. [[Obfuscation|обфускация]]).
