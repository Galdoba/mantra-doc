Параметр указывающий на полноту имеющейся информации о звездной системе. Возможные значения от 0 до 12. 
### **Survey Index Table**
примерное соответсвие чеклисту [[World Creation]]

| Index | Known Data<br>(Соответствующий этап из чек-листа)                                                                                                                                                                              | Survey Class    |
| ----- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | --------------- |
| 0     | Нет данных. Новая запись в базе.                                                                                                                                                                                               | -               |
| 1     | Определен тип и класс первичной звезды<br>(Star System Creation, шаг 1).                                                                                                                                                       | -               |
| 2     | Определена многозвездность системы, орбиты и типы компаньонов (шаг 2). <br>Определено общее количество миров (шаг 3).                                                                                                          | -               |
| 3     | Определены допустимые орбиты и обитаемая зона (шаги 4-5). <br>Миры размещены по орбитам (шаг 6). <br>Определены орбитальные периоды и базовые размеры тел (шаги 7-8).                                                          | Distant Class 0 |
| 4     | Для всех миров определены основные физические параметры: точный размер, состав, плотность, гравитация, массивные спутники (Physical Characteristics, шаги 1-3).                                                                | Class O         |
| 5     | Для всех миров определена Атмосфера (шаг 4) и Гидросфера (шаг 5).                                                                                                                                                              | Class I         |
| 6     | Для всех миров определены параметры вращения, температура, сейсмическая активность (шаги 6-9).                                                                                                                                 | Class II        |
| 7     | Для всех миров определены рейтинги местной жизни, ресурсов и обитаемости (шаги 10-12). <br>Окончательно выбран главный мир (Mainworld) системы (шаг 13).                                                                       | -               |
| 8     | Для главного мира определены социальные характеристики: население, правительство, законы (Social Characteristics, шаги 1-4).                                                                                                   | Class III       |
| 9     | Для главного мира определены технологический уровень, культура, экономика, характеристики звездного порта (шаги 5-8).                                                                                                          | -               |
| 10    | Полная детализация. Завершены все пункты чек-листа. Для главного мира определены военные характеристики и зоны перемещения (шаги 9-10). <br>Данные по второстепенным мирам (secondary worlds) уточнены там, где это применимо. | Class IV        |
| 11    | Данные, полученные в результате прямого наблюдения или посещения, выходящие за рамки процедур генерации (например, подробные карты, полный видовой состав биосферы, детальная политическая история).                           | -               |
| 12    | Исчерпывающие данные, включая скрытые или защищенные сведения (например, секретные базы, неконтактные цивилизации, аномальные явления).                                                                                        | Class V         |

**Логика использования:**
*   Все новые записи получают SI = 0.
*   Для повышения индекса необходимо **полностью и успешно завершить** все этапы генерации, указанные в колонке "Known Data" для целевого индекса.
*   Проверка перехода на новый индекс (например, с SI=3 на SI=4) заключается в верификации того, что все расчеты и определения для миров системы (размер, состав, спутники) выполнены корректно и не противоречат друг другу.
*   Индексы 11 и 12 присваиваются вручную на основе ролевой игры или специальных правил, так как представляют информацию, недостижимую стандартной процедурой генерации.
