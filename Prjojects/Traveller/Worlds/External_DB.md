База данных формирующаяся из данных с сайта https://travellermap.com 
Содержит канонические данные в формате [[t5ss]]. 

Данные получается путем последовательных вызовов через API: 
https://travellermap.com/api/jumpworlds?x=[X]&y=[Y]&jump=[JUMP]
Сылка использует переменные [Глобальных координат] и радиуса колец.
- X - глобальная координата X
- Y - глобальная координата Y
- JUMP - радиус в котором собираются данные относительно глобальных координат {X,Y}

Алгоритм проходить по точками калибрации и собирает все миры классического периода (M1105)

База заполняется [[Импортируемые данные|импортируемыми данными]]
