Ключ-Префикс - это часть имени файла состоящая из блоков разделенных двойным дефисом(`--`).

КП может быть считан с файла или высчитан из проекта.

```go
type Prefix struct {
	ProjectBase string
	SEpMarker string
	ProjectType string
	AutoTaskMarker string
	Separator string
}

```
# Блоки
1. Транслитерированное название (вариативный)
	- исключаем ` (замена*)`
	- конвертируем ` ?? сезон. ?? сери*` в список тэгов `_s??e??`
2. Тип проекта (MOV/SER/TRL)
3. Маркер автоматического запуска (`AUTO` или отсутствует)
